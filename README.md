# 电动车骑行避坑指南

## 可能面临罚款教育的行为

### 罚款项

- 骑电动车带大于12周岁的人
- 骑电动车载人营运
- 在禁止电动车，自行车骑行的路段骑行电动车（50元），例如
    - 高架下
    - BRT 廊道
    - 高架
    - 机动车道主道
- 闯红灯 （20-50元）
- 马路侧边逆向行驶（50元）
- 改装电动车
    - 加装挡雨棚
    - 给电动车套防风套

### 扣车项

- 未上牌（刚买的电瓶车可用发票证明，不会扣车）
- 上牌了，但未随身携带行驶证。

### 拦停教育，不罚款

- 骑电动车未戴头盔

## 避坑方式

- 买车后立即上牌，上牌前随身携带购车发票。
- 不带满12岁的人。
- 道路骑行注意
    - 有可供电瓶车骑行的人行道就走人行道。
    - 有自行车道的走自行车道。
    - 无可骑行人行道，无自行车道，可靠马路边骑行。
    - 不走高架
    - 不走禁止自行车电动车的道路。
    - 不逆向行驶。
    - 不闯红灯
- 车辆注意
    - 不使用车前挡风罩。可穿长裤，围裙等挡风。
